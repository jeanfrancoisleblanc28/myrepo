# Deliverables

Livrables bureautiques et documents finis produits dans ce repo.

## Sous-dossiers

- `html/` — pages HTML autonomes (ex. `organigramme.html`)
- `documents/` — documents Word (.docx), évaluations, rapports rédigés
- (les présentations PPTX restent dans `/presentations/` à la racine pour l'instant)

## Conventions

- Nommer les fichiers en kebab-case : `evaluation-stage-prenom-nom.docx`
- Préfixer les versions par date ISO si besoin : `2026-05-14_rapport-trimestriel.pdf`
- Les sources (scripts, templates) restent à côté du livrable quand c'est pertinent
